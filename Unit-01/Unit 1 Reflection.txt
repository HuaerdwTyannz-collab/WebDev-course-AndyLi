# Unit 1 Reflection

In Unit 1, I learned about the history of the web and the basics of web development. 
I also learned simple HTML structure, basic formatting tags, and how to create a simple HTML page. 
At first, some parts were confusing because there were new terms and code to remember. However, 
after practicing and looking at examples, I understood better how a website is built. 
This unit helped me understand that websites are not only about design, 
but also about structure and organization.
